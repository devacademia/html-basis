Les listes ordonnées

Les listes ordonnées permettent de représenter les éléments d’une liste dans un ordre précis

Example:
Les étapes de pour faire la purée d'avocat:

  1. Decouper les avocats en tranche
  2. Nettoyer les avocats
  3. Les extraire de la peau
  4. Decouper les oignons, la tomate

Les listes non ordonnées

Les listes ordonnées permettent de représenter les éléments d’une liste dont l'ordre n'est pas important.

Exemple: La liste de course

  • Acheter du papier toilette
  • Achetter et nettoyer le poisson
  • Achetter les condiments
